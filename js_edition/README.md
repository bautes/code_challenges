# Prerequisites

1. Install node (v6.5.0 or higher)
2. `npm i`

## Running Specs

`npm run test` or `npm run test -- --watch`


## Running endpoints

`npm run start`

Then

`curl 'http://localhost:3000'`
`curl 'http://localhost:3000?userId=2664'`
`curl 'http://localhost:3000?startDate=2017-02-01&endDate=2017-03-01'`
